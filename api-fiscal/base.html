Documentación de API

Recurso: Vista de todos los reglamentos

URL: ?resource=titles

Descripción: Este recurso proporciona una vista de todos los reglamentos disponibles. Puedes utilizar esta opción para obtener una lista completa de los reglamentos en la API.

Recurso: Vista de un solo reglamento

URL: ?reglament={idReglament}

Descripción: Este recurso permite acceder a la vista de un solo reglamento identificado por su ID. Puedes utilizar esta opción para obtener detalles específicos de un reglamento en particular.

Recurso: Registro de Nuevos Usuarios

URL: ?register=1&name={name}&email={email}&password={password}

Descripción: Este recurso te permite registrar nuevos usuarios en el sistema. Al proporcionar los datos necesarios, como el nombre, el correo electrónico y la contraseña, puedes crear una cuenta de usuario. Una vez registrado, los nuevos usuarios podrán acceder a la aplicación inLibrary.

Recurso: Iniciar sesión

URL: ?login=1&email={email}&password={password}

Descripción: Este recurso te permite iniciar sesión. Al proporcionar los datos necesarios, como el correo electrónico y la contraseña, los usuarios podrán acceder a la aplicación inLibrary.

Recurso: Cargar datos de un usuario

URL: ?data=1&email={email}

Descripción: Este recurso te permite Cargar los datos de un usuario. Al proporcionar el Email.

Recurso: Actualizar datos de un usuario

URL: ?update={idUser}&firstname={firstname}&lastname={lastname}&birthday={birthday}&phone={phone}

Descripción: Este recurso te permite actualizar los datos de un usuario. Al proporcionar el idUser.

Recurso: Busqueda de datos

URL: ?search={busqueda}

Descripción: Este recurso te permite buscar palabras en los articulos.
