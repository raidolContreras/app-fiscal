Documentación de API

Recurso: Vista de todos los reglamentos
Este recurso proporciona una vista de todos los reglamentos disponibles. Puedes utilizar esta opción para obtener una lista completa de los reglamentos en la API.
URL: ?resource=titles
Recurso: Vista de un solo reglamento
Este recurso permite acceder a la vista de un solo reglamento identificado por su ID. Puedes utilizar esta opción para obtener detalles específicos de un reglamento en particular.
URL: ?reglament={idReglament}
Recurso: Registro de Nuevos Usuarios
Este recurso te permite registrar nuevos usuarios en el sistema. Al proporcionar los datos necesarios, como el nombre, el correo electrónico y la contraseña, puedes crear una cuenta de usuario. Una vez registrado, los nuevos usuarios podrán acceder a la aplicación inLibrary.
URL: ?register=1&name={name}&email={email}&password={password}
Recurso: Iniciar sesión
Este recurso te permite iniciar sesión. Al proporcionar los datos necesarios, como el correo electrónico y la contraseña, los usuarios podrán acceder a la aplicación inLibrary.
URL: ?login=1&email={email}&password={password}
Recurso: Cargar datos de un usuario
Este recurso te permite cargar los datos de un usuario. Al proporcionar el correo electrónico.
URL: ?data=1&email={email}
Recurso: Actualizar datos de un usuario
Este recurso te permite actualizar los datos de un usuario. Al proporcionar el idUser.
URL: ?update={idUser}&firstname={firstname}&lastname={lastname}&birthday={birthday}&phone={phone}
Recurso: Búsqueda de datos
Este recurso te permite buscar palabras en los artículos.
URL: ?search={busqueda}
Recurso: Marcar artículos favoritos
Permite marcar artículos como favoritos.
URL: ?article={isArticle}&user={idUser}
